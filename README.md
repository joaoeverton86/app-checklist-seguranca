# App Checklist Segurança v2

Aplicativo de checklist de segurança do trabalho para construction sites.

## Funcionalidades

- Checklist offline (funciona sem internet)
- Sincronização com Google Sheets
- Cadastro de equipamentos e colaboradores
- Exportação CSV
- Assinatura digital do TST

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Ou execute `iniciar.bat` para iniciar o servidor local

## Tecnologias

- HTML5 / CSS3 / JavaScript
- IndexedDB (armazenamento local)
- Service Worker (offline)
- Google Apps Script (sincronização)
